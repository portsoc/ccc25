CCC-Web Plan

Week 1

By the end of this week you will be able to build basic web pages using good semantic markup, showing familiarity with the necessary tools & technologies.

Before the Lecture

  • Watch the introductory scroller.
  • Add Lawson & Sharp's HTML 5 to your digital bookshelf.

Before the Workshop

  • Nothing much: it's week 1 of this sub-module so show us you can turn up!

Lecture Topic

  • Building basic web pages with semantic markup.

Workshop Activity

  • Start the HTML worksheet.
  • Start the Responsive Web Design modules on Basic HTML and Semantic HTML from FreeCodeCamp. You should work through these in your own time to gain experience.

Week 2

By the end of this week you will be able to use a basic style sheet to format a web page.

Before the Lecture

Prepare questions arising from the HTML worksheet.

Before the Workshop

  • Complete all tasks in the HTML worksheet.

Lecture Topic

  • Adding styles to web pages.
  • Extra slides
  • Example Source (with added bits)
  • Live example

Workshop Activity

Demonstrate what you did for Tasks 4+5 of the HTML worksheet.

Starting the Style worksheet.

Week 3

By the end of this week you will be able to use Images in HTML pages, making them accessible and coping with different screen resolutions.

Before the Lecture

Try using images in your pages.

Lecture Topic

Using images properly on the web.

Before the Workshop

Complete tasks in the Style worksheet.

Experiment with HTML & CSS, and make notes on any questions you want to ask in the work session.

Workshop Activity

  • Demonstrate what you did for Task 4 of the Style worksheet.
  • Start the Images worksheet.

Week 4

By the end of this week you will be able to use absolute and relative web addresses for links.

Before the Lecture

N/A

Lecture Topic

  • URLs
  • The URLDemo (Live)
  • The URLDemo (Source)

Before the Workshop

Complete the Images worksheet.

Workshop Activity

  • Demonstrate that you can create HTML with valid images.
  • Start the Addressing worksheet.

Week 5

By the end of this week you will be able to manage and backup your code with git and GitHub.

Before the Lecture

Prepare your brain for a nice relaxing session and find something to make notes with.

Lecture Topic

Cover all the basics of git and GitHub - everything that you'll need to know to be effective when managing coursework files.

Before the Workshop

Do nothing - this week is a bonus - git and GitHub are not on the exam - just bring your brain and something to make notes.

Workshop Activity

Q&A on everything so far, and learning to use git with your own code.
